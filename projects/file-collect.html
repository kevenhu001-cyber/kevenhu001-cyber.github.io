极简文件收集平台

老师不再需要每天手动收作业文件。三分钟创建一个收集任务，剩下的交给系统。

解决的问题：微信群收文件——同名文件会被覆盖、改名丢失来源、收集完成后要一个个点开保存。File-Collect 让教师/组织者只需要三件事就能上线：登录后台 → 新建收集任务 → 把链接发给参与者。参与者无需注册即可上传。剩下的事情全部由系统搞定。

多主题并行收集：同一组织者可同时拥有多个收集主题（如"建模比赛"、"实验报告"、"毕业设计"），每个主题拥有独立提交链接、独立存储空间和独立配置（暂停/恢复、截止时间、速率限制）。

管理后台（单页全功能）：密码登录（默认 000000，可 ADMIN_PASSWORD 环境变量覆盖）→ 生成 32 字节随机 hex token → 持久化到 config/sessions.json（服务器重启登录不丢失，每日自动裁剪至 200 条）。后台功能：查看所有提交者清单及文件大小、按用户筛选、单个删除、批量删除用户（二次确认）、单个用户文件 ZIP 打包、全收集 ZIP 打包下载（Archiver 流式压缩，最高级别 9，无临时文件）、新建/删除收集主题（删除前检查——已有文件则拒绝）、配置暂停/恢复与截止时间（前端实时倒计时，到期自动隐藏上传表单）。

大文件上传：单文件最大 500MB，支持拖拽上传。原始文件名 UTF-8 解码（multer binary → Buffer → utf8 转换，CJK 字符正确保留）。Nginx 层配合 86400s 超时保证大文件不中断。

速率限制：IP 级别可配置每 IP 每小时上传次数上限。纯内存 Map 实现 + 每 5 分钟自动清理过期记录。单实例部署（PM2 instances: 1 确保准确性）。

零数据库架构：全应用无外部数据库依赖——配置存 settings.json、会话存 sessions.json、上传文件存文件系统。原子写入保证一致性。前端全原生 HTML + CSS + JavaScript，零框架、零构建步骤。

部署链路：Nginx 反代（独立配置 /admin /api /collect/ 和静态资源路由）+ Let's Encrypt SSL（Certbot 自动化）+ PM2 进程管理（实例数 1，文件上传场景）。完整 Nginx 配置附于 README。

2024 · 已部署上线 核心栈：Node.js 22 · Express 4 · Multer · Archiver · Nginx + Let's Encrypt + PM2

为什么做这个

每个学期会有老师、活动组织者、社团负责人来找我："能不能做一个简单的工具， 让同学们把作业 / 报名表 / 设计稿传到同一个地方？"

微信群收文件 —— 同名文件会被覆盖、改名丢失来源、收集完成后要一个个点开保存。 这套流程谁经历过谁知道。

目标很明确：教师 / 组织者只需要做三件事就能上线：登录后台 → 新建收集任务 → 把链接发给参与者。 剩下的事情全部由系统搞定。

核心特性

多主题并行收集

同一个组织者可以同时拥有多个收集主题——比如"建模比赛"、"实验报告"、"毕业设计"。 每个主题拥有独立提交链接和独立存储空间。

灵活的窗口管理

每个收集可独立设置截止时间，也可以手动暂停 / 恢复。

大文件上传

单文件最大 500MB，支持拖拽上传。Nginx 层配合长 timeout 以保证大文件不中断。

管理后台

  • 查看所有提交者和文件清单
  • 按用户筛选、单个删除
  • 一键 ZIP 打包下载

速率限制 & 登录持久化

内置 IP 级别的上传频率控制。Session 持久化存储，服务器重启或浏览器关闭后无需重新登录。

技术栈

组件技术
后端Node.js 22 · Express 4
反向代理Nginx 1.24
SSLLet's Encrypt · Certbot
前端原生 HTML + CSS + JavaScript（无框架）
文件存储本地文件系统
进程管理PM2
关键依赖multer · archiver · cookie-parser

学到了什么

  1. 小项目反而能教会你最具体的工程素养。文件上传、Nginx 反代、PM2 重启、SSL 续期 —— 平时不写一遍真的不会彻底搞懂。
  2. "无框架"不等于"原始"。这个项目前端是纯 HTML + CSS + JS，但依然有清晰的页面结构和良好的 UX。
  3. 安全要落到"最小改动可防御的点"。改 ADMIN_PASSWORD 环境变量 → 上线前必须改。够用就行，别过度设计。
  4. 速率限制不要等出事了再加。某个收集被一台机器反复刷之后，才意识到 IP 维度的基础限流该作为默认配置出现。

部署链路：git clone → npm install → pm2 start → 配 Nginx → certbot 拿证书，五分钟以内。
